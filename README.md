🏆 Final Project – FIFA World Cup 2026
A full‑stack web application built with Next.js and Flask, integrating four external FIFA World Cup APIs to deliver real‑time data about teams, stadiums, matches, and routes. The project combines a modern frontend with a custom backend to create a complete, production‑style architecture.

📌 Overview
This project is a complete web platform that allows users to explore detailed information about the 2026 FIFA World Cup. It includes:

A Next.js frontend for dynamic UI, routing, and API consumption

A Flask backend providing custom REST endpoints

Integration with multiple external APIs for real data

Full communication between frontend ↔ backend using JSON

The goal is to demonstrate a real full‑stack workflow, modern development practices, and API orchestration.

🧩 Main Features
Teams Explorer  
Browse all World Cup teams with real data from external APIs.

Stadiums Viewer  
Display stadium details, capacity, and location.

Match Schedule  
View upcoming matches, dates, and participating teams.

Route Finder  
Fetch routes and travel information to stadiums.

Custom Backend Features
